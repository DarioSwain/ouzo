Welcome to Ouzo.

Start using it right now!

$ composer.phar create-project letsdrink/ouzo-app:dev-master myproject

Check out the Tutorial to learn more.

Features:

  • Build-in orm

    No more awkward sql. Get everything you need with a simple and elegant query builder:

    User::where(['login' => 'john'])->fetch();
  • RESTful Routing

    Route::resource('users');

    This simple code will map all REST actions for your resource.

  • Database schema management

    Ouzo uses ruckusing-migrations for schema management.

  • Testing framework

    Controllers, Models tests

  • Utility classes

    Ouzo comes with plenty of utility classes that will help you deal with strings, arrays and more.

  • Translations
  • Database agnostic

    Supports postgres, mysql, mariadb. You can easily write an Adapter for a db you need.

  • Waterproof

    This is the only waterproof framework on the market. Let's see a quick comparison:

    Zend Symfony Ralavel Ouzo
    Waterproof No No No Yes
